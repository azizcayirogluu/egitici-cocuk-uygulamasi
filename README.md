## 🌟 Proje Hakkında

**Eğitici**, çocukların temel kavramları eğlenerek öğrenmesi için tasarlanmış renkli ve etkileşimli bir mobil uygulamadır. Canlı animasyonlar, arka plan müzikleri, sesli yönlendirmeler (TTS) ve oyunlarla dolu bu uygulama, çocukların zihinsel gelişimini desteklemeyi amaçlar.

Uygulama tamamen Türkçe olarak geliştirilmiştir.

---

## 🚀 Özellikler (Eğitim Modülleri)

Uygulama içerisinde çocukların farklı alanlarda kendilerini geliştirebileceği 6 farklı modül bulunmaktadır:

*   🌤️ **Mevsimler:** Dört mevsimi görsel ve işitsel öğelerle öğrenme.
*   🔢 **Sayılar:** Sayıları tanıma ve sayma pratikleri.
*   🎨 **Renkler:** Ana ve ara renkleri görsel hafıza ile öğrenme.
*   🦁 **Hayvanlar:** Hayvanları isimleri, görselleri ve sesleriyle tanıma.
*   🖌️ **Boyama:** Çocukların yaratıcılıklarını konuşturabilecekleri serbest boyama ekranı (Taşırma engellemeli).
*   🧩 **Eşleştirme:** Hafızayı güçlendiren, eğlenceli kart eşleştirme oyunu.

---

## 🛠️ Kullanılan Teknolojiler ve Paketler

Projenin geliştirilmesinde aşağıdaki Flutter paketleri kullanılmıştır:

*   `audioplayers`: Arka plan müzikleri ve efekt sesleri için.
*   `flutter_tts`: Metin okuma (Text-to-Speech) desteğiyle çocuklara sesli geribildirim sağlamak için.
*   `floodfill_image`: Boyama modülünde akıllı boyama (dokunulan alanı boyama) özelliği için.
*   `cupertino_icons`: iOS tarzı ikonlar için.

## ⚙️ Kurulum ve Başlangıç

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyin:

### Ön Koşullar
- Flutter SDK (Sürüm 3.11.1 veya üzeri)
- Android Studio / VS Code
- Aktif bir Emülatör veya fiziksel test cihazı

### Adımlar

1. **Projeyi Klonlayın:**
   ```bash
   git clone https://github.com/kullaniciadi/egitici.git
   ```

2. **Proje Dizinine Gidin:**
   ```bash
   cd egitici
   ```

3. **Gerekli Paketleri Yükleyin:**
   ```bash
   flutter pub get
   ```

4. **Uygulamayı Çalıştırın:**
   ```bash
   flutter run
   ```

## 🎵 Medya ve Sesler

Uygulamanın ayarlar menüsünden arka plan müziği açılıp kapatılabilir. Projede kullanılan tüm müzik ve görseller `assets/` klasörü altında barındırılmaktadır. 
- Müzikler: `assets/sounds/`
- Görseller: `assets/Boyama/` ve diğer dizinler.
